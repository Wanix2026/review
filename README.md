# Review — Claude Code 医学生复习资料整理技能

为医学生系统整理课程复习资料。从大纲、教学日历、PPT/PDF课件、复习重点、题库等分散源文件中提取知识点，以**大纲为骨架、教学日历为时间轴、PPT为主要内容源**，生成结构化复习笔记。

## 设计理念

- **大纲是唯一的真理源** — 所有输出内容必须能回溯到大纲中的某一行
- **三源交叉验证** — 大纲、教学日历、PPT 三者逐章比对，不一致处显式标注
- **不生成内容** — 只整理、提取、归类，不凭空编写知识点
- **缺口显式标注** — 资料缺失处明确标记，不与正常内容混淆

## 快速开始

### 安装

```bash
# 克隆到 Claude Code 技能目录
git clone https://github.com/Wanix2026/review.git ~/.claude/skills/review

# 安装依赖
pip3 install python-pptx python-docx openpyxl pdfminer.six
```

### 使用

在 Claude Code 中输入：

```
/review /path/to/课程目录
```

或直接说"帮我整理这门课的复习资料"，技能会自动加载。

### 双模式

| 条件 | 行为 |
|------|------|
| **用户提供了题型分布** | 按题型驱动格式（名词解释/A1/A2/B1/简答/判断） |
| **用户未提供题型**（默认） | 纯知识点整理模式，不区分题型 |

## 执行流程

```
Phase 0 · 文件发现与分类
  ├─ 扫描目录 → 自动分类（大纲/日历/PPT/复习重点/题库）
  ├─ 内容特征验证 + 重复检测
  ├─ 🔒 闸门检查（大纲+教学日历必须存在）
  ├─ 四层章节映射（编号→内容采样→关键词→日历桥接）
  └─ 环境依赖预检

Phase 1 · 全量提取（增量+缓存）
  ├─ SHA256 文件指纹
  ├─ 格式诊断（PDF文字版/扫描版/加密 | DOCX/DOC/PPTX/XLSX）
  ├─ 完整性自检 + 乱码检测
  └─ .course-cache/ 缓存

Phase 2 · 三源交叉比对
  ├─ 16 状态决策表（大纲×日历×PPT）
  ├─ 缺口分级（🔴致命 🟡警告 🔵可选）
  └─ 三选一响应（补文件/确认缺失/移除此章）

Phase 3 · 复习资料生成
  ├─ 三段式章节（大纲要求→内容要点→可能的考点）
  ├─ HTML 输出（侧栏目录+滚动监听+移动端+打印样式）
  └─ 可选题库处理（Web平台/PDF题库→清洗→.docx导出）
```

## 输出格式

- **Markdown** — 源文件，方便后续修改
- **HTML** — 侧栏目录跳转、移动端适配、打印优化
- **PDF** — 打印版（可选）
- **Word (.docx)** — 题库导入格式（可选）

## 适用场景

- 医学院期末考试复习资料整理
- 课程PPT+大纲→结构化知识笔记
- 在线题库平台导出（人卫/学习通）
- 考前冲刺提纲生成
